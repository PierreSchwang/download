What is FastAsyncVoxelSniper?
FAVS is a replacement for the abandoned VoxelSniper. Using FAVS over VS allows you to restrict it to regions and speed it up.

  1. Download FastAsyncVoxelSniper (FAVS)

    Please ensure the correct platform & version is selected above.
    Versions labeled as "Spigot" run on Paper as well.
    Versions below 1.14 are classified as legacy and receive limited to none support.

  2. Download any optional dependencies

  3. Configure the plugin + Give players the required permissions

    • You can find the config.yml file at: /FastAsyncVoxelSniper/config.yml
    • General FAVS Permissions
